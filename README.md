# 🧙‍♂️ BoostWiz   
> 스나이퍼 팩토리 프론트엔드 프로젝트 캠프에서 진행한 "[KT Wiz 홈페이지](https://www.ktwiz.co.kr/) UI/UX 개선 프로젝트"를 마친 후, 개인적으로 리팩토링을 진행하는 레포지토리입니다.

- 👪 기관: 스나이퍼 팩토리, KT
- 📆 기간: 2024.12.02 ~ 2025.01.17(7주) + 개인 리팩토링(2025.02.12 ~)

<br/>


## 💡 목표 및 개요
기존 야구 사이트의 인터페이스를 현대적이고 직관적인 사용자 경험으로 리뉴얼하여, 팬들에게 더욱 몰입도 높은 서비스를 제공

<br/>
